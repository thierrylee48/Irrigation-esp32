# irrigation-esp32
use esp32 controls the gate closing based on water level sensed by ultrasonic distance sensor.

## Development Environment
- Arduino IDE Version: 2.3.8
- Board Package: esp32 by espressif Systems
  - Purpose: Allow Arduino IDE to understand and use ESP32
  - Version: 3.3.7

## External Libraries / Services Used
- ESP32Servo
  - Purpose: Control servo motor connected to ESP32
  - Version: 3.1.3
